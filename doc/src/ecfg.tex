The \gls{eCFG} is the internal way for \flatac to represent the \tsl{Frama-C} \gls{CFG} plus our own informations such as :
\begin{itemize}
	\item Abstract interpretation
	\item Counter automata label
	\item Value analysis
	\item Statement
\end{itemize}

\image{img/ecfg}{1}

In principle, you \tbf{do not have to access directly to the \gls{eCFG}}.
