This class defines an object type that allows to store and operate manipulation on couter automata. Namely : _ Defining a set of initial and a set of final control state. _ Storing the set of transitions. _ Computing the set of control state from the set of transitions. _ Adding a transition. _ Pretty printing a counter automaton into FLATA syntax.
