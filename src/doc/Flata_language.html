In this file, we define the function that allows to check whether an expression of the intermediate language is expressible using flata's grammar.

The intermediate language allows to express things like "x'=x*y", for instance, which is of course not possible to express using the grammar of FLATA.

An arithmetic operation is expressible using a FLATA term if it contains no monomial whose degree is greater than one. Here, by polynomial variables, we mean counters.
